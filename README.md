# DateFruit ANN Prediction System

## Overview

The **DateFruit ANN Prediction System** is a deep learning project that uses Artificial Neural Networks (ANN) to analyze and predict characteristics of date fruits.

The project applies both **classification and regression techniques** using neural networks to study patterns within the Date Fruit dataset. This system demonstrates how deep learning models can be used for agricultural data analysis and predictive modeling.

---

## Machine Learning Type

This project uses **Supervised Learning**.

Two different tasks are implemented:

### ANN Classification

Used to classify date fruits into different categories based on their characteristics.

### ANN Regression

Used to predict continuous values related to date fruit properties.

---

## Dataset Description

The **Date Fruit dataset** contains multiple features describing the physical and chemical characteristics of date fruits.

Typical features may include:

* Fruit size
* Shape characteristics
* Color features
* Texture properties
* Chemical composition
* Other agricultural measurements

These features help build predictive models for classification and regression tasks.
